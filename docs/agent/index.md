# Agent 工程

## 什么是 Agent

Agent 是基于大模型能力构建的任务执行系统。

相比传统问答模式，Agent
不仅生成文本，还能够理解目标、调用工具、执行流程并根据结果调整任务。

------------------------------------------------------------------------

## 核心组成

一个完整 Agent 系统通常包括：

-   大语言模型；
-   状态管理；
-   工具调用；
-   流程编排；
-   记忆机制。

------------------------------------------------------------------------

## 工程设计

实际应用中，需要关注：

-   任务拆解；
-   工作流设计；
-   多轮执行；
-   上下文管理；
-   异常处理。

常见工程框架：

-   LangGraph；
-   LangChain。
