# MCP与工具调用设计

## 工具调用

大模型本身不能直接操作外部系统，需要通过工具扩展能力。

例如：

-   查询数据库；
-   调用接口；
-   搜索信息。

------------------------------------------------------------------------

## MCP作用

MCP提供统一方式，让模型连接外部工具和数据。

------------------------------------------------------------------------

## Skill、Tool与Agent

Tool：

执行具体能力。

Skill：

描述任务执行方式。

Agent：

负责理解任务并组织执行流程。

三者共同组成智能应用系统。
