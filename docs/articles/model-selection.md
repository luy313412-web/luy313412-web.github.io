# AI应用中的模型选择

## 不同模型承担不同职责

实际系统中通常不是单一模型完成所有任务。

例如：

-   通用模型负责复杂推理；
-   小模型负责分类；
-   Embedding模型负责检索。

------------------------------------------------------------------------

## 选择原则

考虑：

-   效果；
-   成本；
-   响应速度；
-   部署方式。

------------------------------------------------------------------------

## 工程理解

模型选择不是追求最大模型，而是根据业务需求选择合适能力。
