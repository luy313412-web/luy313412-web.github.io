# 从Demo到生产级AI系统

## Demo的问题

简单Demo通常只关注：

模型能否回答。

生产系统需要关注：

-   稳定性；
-   成本；
-   权限；
-   监控；
-   评测。

------------------------------------------------------------------------

## 生产化能力

包括：

-   日志追踪；
-   异常处理；
-   模型降级；
-   数据治理；
-   效果优化。

------------------------------------------------------------------------

## 个人理解

AI应用最终需要结合模型能力和软件工程能力。
