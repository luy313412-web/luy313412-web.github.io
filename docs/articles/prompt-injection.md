# Agent安全与Prompt Injection防护

## 风险来源

智能体能够调用工具后，攻击面增加。

风险包括：

-   恶意输入；
-   指令覆盖；
-   工具滥用。

------------------------------------------------------------------------

## 防护方式

输入层：

-   内容检测；
-   风险识别。

工具层：

-   权限控制；
-   工具白名单。

输出层：

-   内容校验；
-   敏感信息过滤。

------------------------------------------------------------------------

## 工程理解

Agent安全不是单一Prompt解决，而需要系统级防护。
