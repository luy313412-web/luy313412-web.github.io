# 大模型工程

## 模型能力

大模型提供：

-   文本理解；
-   推理能力；
-   内容生成；
-   代码辅助。

但实际应用需要结合工程系统。

------------------------------------------------------------------------

## 应用开发

常见能力包括：

-   Prompt设计；
-   结构化输出；
-   Function Calling；
-   工具调用。

------------------------------------------------------------------------

## 上下文管理

长任务需要考虑：

-   历史信息保存；
-   摘要压缩；
-   Token控制；
-   重要信息提取。

------------------------------------------------------------------------

## 模型优化

包括：

-   知识蒸馏；
-   模型微调；
-   推理优化；
-   成本控制。

------------------------------------------------------------------------

## 工程实践

模型选择需要综合考虑：

-   效果；
-   成本；
-   响应速度；
-   部署方式。
