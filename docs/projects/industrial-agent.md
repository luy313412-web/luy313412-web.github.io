# 工业设备运维 Agent

## 项目背景

面向工业设备运维场景，结合大模型、知识检索和智能体技术，辅助现场人员进行故障分析、维修建议和设备状态管理。

## 业务目标

实现：

-   故障问题分析；
-   维修知识查询；
-   工单辅助处理；
-   设备健康状态分析。

## 系统设计

整体流程：

用户请求 → API服务 → 智能体编排 → 业务子流程 → 模型与知识库 → 返回结果。

核心采用流程编排与模型决策结合的方式，提高任务处理稳定性。

## 核心技术

-   LangGraph流程编排；
-   RAG知识增强；
-   工具调用；
-   状态管理；
-   模型结构化输出。

## 工程实践

重点关注：

-   上下文管理；
-   异常处理；
-   成本控制；
-   效果评估。
