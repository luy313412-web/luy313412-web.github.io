# RAG 工程

## 什么是 RAG

RAG（检索增强生成）通过引入外部知识，提高大模型回答的准确性和可靠性。

------------------------------------------------------------------------

## 完整流程

``` text
数据处理

↓

文档切块

↓

向量化

↓

检索召回

↓

重排序

↓

大模型生成
```

------------------------------------------------------------------------
