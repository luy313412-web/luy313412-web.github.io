## Agent工程重点

一个可靠 Agent 需要：

### 状态管理

保存任务进度和上下文。

### 工具管理

限制模型可使用的能力范围。

### 错误处理

面对模型错误和工具失败时能够恢复。

### 评测体系

持续发现问题并优化。
