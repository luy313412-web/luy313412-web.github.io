# 为什么 Agent 成为 AI 应用的重要方向

## 从模型调用到任务执行

传统大模型应用通常是：

用户输入

↓

模型生成

↓

返回结果

而智能体系统进一步引入：

-   任务理解；
-   流程规划；
-   工具调用；
-   状态管理；
-   结果反馈。

使模型能够参与更复杂的任务执行。

------------------------------------------------------------------------

## Agent并不意味着完全自主

不同业务需要不同程度的自主能力。

对于工业、企业流程：

更适合：

固定流程 + 模型决策

保证：

-   可控；
-   可追踪；
-   可维护。

对于探索性任务：

可以采用更强的自主规划能力。

------------------------------------------------------------------------

## 个人理解

Agent
的价值不是让模型无限自由运行，而是让模型在合理约束下完成更复杂任务。
