# 固定流程和自主 Agent 如何选择

## 固定流程

适合：

-   工业流程；
-   企业审批；
-   规则明确业务。

优势：

-   可控；
-   稳定；
-   容易维护。

------------------------------------------------------------------------

## 自主 Agent

适合：

-   信息收集；
-   编程任务；
-   探索性问题。

优势：

-   灵活；
-   适应复杂任务。

------------------------------------------------------------------------

## 选择原则

不是自主程度越高越好。

应该根据业务特点选择：

确定性强的任务采用流程编排。

开放性强的任务增加自主决策能力。
